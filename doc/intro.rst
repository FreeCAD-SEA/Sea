========
FreeCAD
========
FreeCAD is a parametric 3D modeler. Parametric modeling allows you to easily
modify your design by going back into your model history and changing its
parameters. FreeCAD is open-source (LGPL licensed) and completely modular,
allowing for very advanced extension and customization.

FreeCAD can be accessed through Python by various means.

========
Python
========
Python is a general-purpose, high-level programming language whose design
philosophy emphasizes code readability. Its syntax is said to be clear and
expressive. Python has a large and comprehensive standard library.
